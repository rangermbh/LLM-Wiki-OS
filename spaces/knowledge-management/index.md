# Knowledge Management Domain Index

## Concepts

<!-- Core PKM concepts. See wiki/concepts/ -->

- [[wiki/concepts/note-atomicity|Note Atomicity]] — 笔记原子性：每一条笔记只包含一个独立概念，使知识单元可跨上下文复用和链接。Principle-type，KM Domain 首个锚点概念
- [[wiki/concepts/zettelkasten-method|Zettelkasten Method]] — 卡片盒方法：以原子笔记为基本单元、通过密集链接和序列编号构建知识网络的方法论。Method-type，依赖 Note Atomicity 作为前置原则
- [[wiki/concepts/emergent-organization|Emergent Organization]] — 涌现组织：当定义良好的原子知识单元通过局部链接决策相互连接，知识结构和主题集群从链接积累中自发形成的现象。Result-type，依赖 Note Atomicity（单元质量）和 Zettelkasten Method（链接机制）作为前置条件
